# KOKOFriends

## Features

- 無好友
- 好友列表無邀請
- 好友列表含邀請好友
- 下拉刷新
- 點擊搜尋框，畫⾯上推至搜尋框置頂至navigationBar下⽅
- 邀請列表⽀援縮合操作

## Getting Started

1. Clone this repository
2. Open `KOKOFriends.xcodeproj`
3. Build and run on simulator
